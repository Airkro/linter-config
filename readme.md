# stylelint-config-airkro

stylelint config for personal use

## Installation

```js
npm install stylelint-config-airkro -O

// or

yarn stylelint-config-airkro -O
```

## Usage

Add following settings to your `.stylelintrc`:

```json
{
  "extends": "stylelint-config-airkro"
}
```
